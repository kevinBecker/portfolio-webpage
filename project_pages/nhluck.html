Sports Game Predictor

This automated data analysis system leverages publicly available APIs to predict NHL game outcomes. The program features:

  • Real-time game schedule monitoring
  • Automated data collection for relevant teams
  • Custom prediction algorithm implementation

The system streamlines the prediction process by automatically identifying upcoming games, gathering relevant team statistics, and applying a specialized algorithm to forecast game results. Analysis of the prediction accuracy is currently in progress.
