Distributed Simulation Cluster

This project, my master's thesis, features a unique implementation where all necessary code is written in Bash. This design choice eliminates external dependencies, enabling the software to run seamlessly on Ubuntu, Raspbian, or MacOS. The lightweight architecture allows deployment on Raspberry Pi devices. The software suite automatically handles code management, including downloading, compiling, running, of code from any Git or SVN server. Automatic uninstallation and removal of repositories is included as well. It includes a set of tools for viewing and managing ongoing simulations in a queue.

The project has been accepted for presentation at the Oceans 2024 conference. You can view the conference paper here or my complete master's thesis here. During development, this project supported another MIT master's thesis and contributed to multiple research papers. As of 2025, it remains actively used in the laboratory.
