My Projects

Distributed Simulation Cluter.

This project, my master's thesis, has some unique attributes. In particular, all the necesary code is written in bash. This eliminates any dependencies, allowing it to run on Ubuntu, Raspbian, or MacOS. Being lightweight as possible allows the software to run on a Raspberry Pi. Additionally, this software automatically downloads, compiles, runs, (and deletes) code from any git or SVN server. The software suite comes with a set of tools to view and manage all ongoing simulations in a queue. View the paper, which was accepted to the Oceans 2024 conference here, or my full master's thesis here.

Nonlinear Adaptive Controller

I wrote a nonlinear adaptive controller for an autonomous surface vehicle. The controlelr may be configured as a PD controller with a sliding variable to integrate about, or it may be configured as a sliding-mode controller. See the final report here.

Simulated Kitchen Assistant Robot

This simulated robot combines high-level autonomy and low-level path planning into one project. This robot uses PDDL (reference here) to determine what actions to take in its library of possible actions. The robot also uses an RRT to explore the possible configuration-space to reach the desired object. The scope did not implement path object avoidance and path optimization together on one trajectory, but they were implemented separately. See the final videos here.

NHL Game predictor

This fully-automated email-list and data analysis program pulls from free, online APIs to predict the outcomes of NHL games. This program starts by finding what games are happening today, downloads the necessary data for those teams, and runs a special prediction algorithm to predict the results. Analysis of these results is in progress.
